"Connection Failed"

1. Make sure server is running: node server.js
2. Check terminal for errors
3. Visit http://localhost:3000/api/health
4. Run npm install if dependencies missing

"Invalid API Key"

1. Key copied correctly? No extra spaces
2. Key has "Assets" permission with Read+Write
3. Key not expired/deleted on dashboard

"403 Forbidden"

Your server's IP is not in the API key's whitelist.
Fix: Add 0.0.0.0/0 to accepted IPs in Creator Dashboard.

"Still Processing"

Large files take time. Wait 1-2 minutes then check your Roblox Inventory → Models.

Still Not Working?

Open browser DevTools (F12) → Console tab for errors. Also check terminal running the server.

Is my API key safe?

Yes. Stored only in YOUR browser localStorage. Server uses it for the single request only.

What files supported?

.rbxm and .rbxmx up to 50MB.

Can I upload to Group?

Yes. Select "Group" type and enter Group ID.

Where does asset appear?

Your Roblox Inventory → Models, and Toolbox in Studio.
